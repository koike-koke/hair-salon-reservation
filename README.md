# 美容室向け予約サイト
**〜 LINE認証を採用した、美容室予約システム 〜**

## 🌟このアプリを作った理由<br>
　フリーランスとして活動する友人の美容師が、SNSのDMで予約を受け、手作業で予約管理アプリに入力しているということを知り、その手間を解消するために開発しました。

## 🍀　このアプリで工夫したポイント

### １.論理削除の導入
　予約をただ消すのではなく「キャンセル済み」という履歴を残すようにしました。
「誰がいつキャンセルしたか」を後で確認できるようにすることで、予約が急に消えて困るのを防ぎ、今後の分析にも役立てられるようにこだわりました。

### ２.正確な予約重複チェックロジック
同じ時間に予約が重ならないよう、開始・終了時間を自動で判別する機能を組み込みました。キャンセルされたデータは除外して判定することで、常に正しい空き枠が表示されるようにしています。

### ３.LINEログインによる認証
会員登録の手間を減らすため、使い慣れたLINEでそのままログインできるようにしました。パスワード管理の負担をなくすだけでなく、将来的な公式LINE連携も見据えた設計にしています。

## 🛠使用技術
バックエンド: Java 21 / Spring Boot 4.0.1<br>
セキュリティ: Spring Security (OAuth 2.0 / LINE Login)<br>
データベース: MySQL <br>
ORM: Spring Data JPA<br>
フロントエンド: Thymeleaf / HTML5 / CSS3<br>
ビルドツール: Maven<br>
その他: Lombok<br>
